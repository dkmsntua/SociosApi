Socios objects

Objects Description

The following table displays the objects of the Socios Object Model, including fields' descriptions and examples.

  • MAIN OBJECTS
    • Person
    • MediaItem
    • Activity
    • Comment
    • SocialNetwork
  • SECONDARY OBJECTS
    • ObjectId
    • Address
    • Name
    • License
  • FILTERS
    • AreaFilter
    • AddressFilter
    • LocationFilter
    • DateTimeFilter
    • PersonFilter
    • ActivityFilter
    • MediaItemFilter
  • HELPER OBJECTS
    • PersonsContainer
    • MediaItemsContainer
    • ActivitiesContainer
    • CommentsContainer
    • ObjectIdContainer
    • SociosException
    • Fault Codes
JSON | XML

The Person object represents a user's profile on a social network. It contains the user's account and profile information.

id String The unique identifier assigned to the user by the social network
sn SocialNetwork The social network
aboutMe String A short description of the user
addresses List<Address> A list of addresses associated with the user. Depending on the social network, this list can include the current location, the hometown or the places where the user has lived
birthday XMLGregorianCalendar The user's birthday
currentLocation Address The user's current location
username String An alias for the user. Depending on the social network this can correspond to the username, the display name, or the screen name
email String The user's email address
gender String The user's gender
name Name The user's full name
photos List<String> Urls of photos associated with the user's account, such as banners, covers or background pictures
profileUrl String The url that points to the user's profile
memberSince XMLGregorianCalendar The date the user's account was created
thumbnailUrl String The user's profile picture
utcOffset Integer The user's current timezone offset from UTC
numFriends Integer The total number of users this user is connected to with a bidirectional link
inDegree Integer The total number of users following or subscribed to this user
outDegree Integer The total number of users this user follows or is subscribed to
JSON | XML

The MediaItem object represents a post published by a user in a social network. It can refer to a video, image or text post.

id String The unique identifier assigned to the media item by the social network
sn SocialNetwork The social network
title String The title of the media item
created XMLGregorianCalendar The date that the media item was published
thumbnailUrl String The url of the thumbnail for the video or the image
description String The text content of the media item, or a suitable description
duration Integer The video's duration in seconds
location Address A location associated with the media item
language String The language of the media item
license License The license attached to the media item
fileSize Long The size of the media item file (image or video) in bytes
rating Double Average rating of the media item (range depends on the social network)
numRatings Integer Total number of people who have rated the media item
numPositiveVotes Integer Total number of positive votes for the media item (likes, plus ones etc)
numNegativeVotes Integer Total number of negative votes for the media item (dislikes)
numComments Integer Total number of comments on the media item
numViews Integer Total number of times the media items has been viewed
numResharings Integer Total number of times the media item has been shared
numFavorites Integer Total number of times the media item has been favorited
tags String A string containing a list of comma separated tags attached to the media item
taggedPeople String A string containing a list of comma separated ids of users associated with the media item
type MediaItemType The type of the media item. Valid values are: IMAGE, VIDEO and TEXT
url String The url that points to the media item
userId String The id of the publisher of the media item
comments List<Comment> The list of comments of the media item
JSON | XML

The Activity object contains information about an action performed by a user in a social network.

id String The unique identifier assigned to the activity by the social network
sn SocialNetwork The social network
created XMLGregorianCalendar The date that the activity took place
title String The title of the activity
description String A description for the activity
location Address The location where this activity occurred
actorId String The id of the user who performed the activity
objectType ActivityObjectType The type of the object(s) associated with the activity. Valid values are: PERSON, MEDIAITEM, ACTIVITY and OTHER
mediaItems List<MediaItem> Media items associated with the activity
persons List<Person> Person objects associated with the activity
activities List<Activity> Activities associated with the activity
JSON | XML

The Comment object represents a comment on a media item published in a social network.

id String The unique identifier assigned to the comment by the social network
sn SocialNetwork The social network
created XMLGregorianCalendar The date that the comment was published
description String The content (text) of the comment
userId String The id of the commenter
username String The username of the commenter
numPositiveVotes Integer Total number of positive votes for the comment (likes, plus ones etc)

The SocialNetwork is an enumeration of the supported social networks.

FLICKR
FACEBOOK
TWITTER
YOUTUBE
DAILYMOTION
GOOGLEP
INSTAGRAM
JSON | XML

The ObjectId is a wrapper that defines an object in a specific social network.

id String The id of the object
socialNetwork SocialNetwork The social network
JSON | XML

The Address object contains details about a location.

country String The location's country
extendedAddress String Detailed description of the location that may include street name, number, city etc
latitude Double The latitude of the location's position
longitude Double The longitude of the location's position
postalCode String The postal code of the location
region String The region of the location
streetAddress String The street name and number
JSON | XML

The Name object contains information about the user's full name.

firstName String The user's first name
lastName String The user's last name
additionalName String An additional handle such as a nickname, middle name, etc
fullName String A detailed version of the full name that may include prefixes, suffixes, academic or honorary titles etc
JSON | XML

The License object contains basic information about the license attached to a media item.

licenseType LicenseType The type of the license. Valid values are: "CC", for Creative Commons license, and "ALL", for all other types
name String The name of the license
url String The url where detailed information about the license can be found

The AreaFilter object is a set of parameters intended to limit a search within an area.

latitude Double The latitude of the center of the search area
longitude Double The longitude of the center of the search area
radius Double The radius for defining the search area. If not provided, a default value is used, depending on the social network

The AddressFilter object is a set of parameters that can be used for address based searches.

country String The country
postalCode String The postal code
region String The region

The LocationFilter object is a pair of filters allowing location based searches.

addressFilter AddressFilter The filter to be used for address based searches
areaFilter AreaFilter The filter to be used for area based searches

The DateTimeFilter object is a set of parameters allowing searches within a specific time frame.

from XMLGregorianCalendar The start date of the time frame
to XMLGregorianCalendar The end date of the time frame

The PersonFilter object is a set of parameters that can be used to search for users.

keywords List<String> A list of keywords to be used as search terms
sns List<SocialNetwork> The list of social networks to include in the search

The ActivityFilter object is a set of parameters that can be used to search for activities.

keywords List<String> A list of keywords to be used as search terms
language String The language of the activities
sns List<SocialNetwork> The list of social networks to include in the search

The MediaItemFilter object is a set of parameters that can be used to search for media items.

created DateTimeFilter A filter that defines a time frame for the search
keywords List<String> A list of keywords to be used as search terms
location LocationFilter A filter that defines the location of the search
language String The language of the media items
licenseType LicenseType The type of the license. Valid values are: "CC", for Creative Commons license, and "ALL", for all other types
sns List<SocialNetwork> The list of social networks to include in the search

The PersonsContainer object contains the results of a request for users.

persons List<Person> A list of Person objects
exceptions List<SociosException> A list of the exceptions that occurred during the request

The MediaItemsContainer object contains the results of a request for media items.

mediaItems List<MediaItem> A list of MediaItem objects
exceptions List<SociosException> A list of the exceptions that occurred during the request

The ActivitiesContainer object contains the results of a request for activities.

activities List<Activity> A list of Activity objects
exceptions List<SociosException> A list of the exceptions that occurred during the request

The CommentsContainer object contains the results of a request for comments.

comments List<Comment> A list of Comment objects
exceptions List<SociosException> A list of the exceptions that occurred during the request

The ÎŸbjectIdContainer contains the id of a social network object, returned as a result of a request.

objectId ObjectId The id of the object
exceptions List<SociosException> A list of the exceptions that occurred during the request
JSON | XML

The SociosException contains information about an error that occurred during a request.

message String A short message about the error
description String A description of the error
faultCode Integer A number indicating the error type
actorId String The id of the object that caused the exception
socialNetwork SocialNetwork The social network

The Fault Codes of the socios exceptions.

400 "No valid input was detected"
404 "The resource you requested was not found"
500 "Something went wrong"
501 "This method is not implemented for this particular social network"
